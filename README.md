# 🏭 총괄생산계획 시각화 웹앱

> 원예장비 제조업체를 위한 **Pyomo 기반 LP/IP 최적화** 총괄생산계획(APP) 수립 및 시나리오 비교 웹앱

---

## 📋 프로젝트 개요

수요·운영 파라미터를 입력하면 **Pyomo**가 자동으로 비용 최소화 생산계획을 산출하고, 결과를 대시보드로 시각화합니다.  
다양한 가정(잔업 허용, 외주 단가 변경 등)을 시나리오로 저장·비교해 최적안을 선택할 수 있습니다.

- **강의자료**: 홍익대학교 스마트제조 — 총괄생산계획 (Chunghun Ha)
- **기준 예제**: 원예장비 제조업체 6개월 계획 (LP 최적해: 422,275천원)

## ⚙️ 설치 및 실행

### 1. 환경 요구사항

- Python 3.9 이상
- GLPK 솔버 (별도 설치 필요)

### 2. GLPK 솔버 설치

| OS | 명령어 |
|----|--------|
| Windows | [winlibs.com](https://winlibs.com/) 또는 conda 사용 권장 |
| macOS | `brew install glpk` |
| Ubuntu/Debian | `sudo apt-get install glpk-utils` |

**conda 환경(권장):**
```bash
conda install -c conda-forge glpk
```

### 3. 패키지 설치

```bash
pip install -r requirements.txt
```

### 4. 앱 실행

```bash
streamlit run app.py
```

브라우저에서 `http://localhost:8501` 자동 접속

## 🚀 주요 기능

### 1️⃣ 수요 및 계획 입력
- 계획 기간(2~24개월/주) 및 타임버킷 설정
- 기간별 수요를 그리드 형태로 직접 입력
- 합계·평균 자동 표시 및 입력값 유효성 검증

### 2️⃣ 운영 파라미터 설정
- 초기 종업원 수, 초기/목표 재고 설정
- 작업일수, 작업시간, 표준시간, 초과시간 상한 설정
- 강의록 기본값 템플릿 원클릭 적용
- 인건비, 고용/해고비, 재고유지비, 부재고비, 재료비, 하청비 입력

### 3️⃣ 총괄계획 자동 산출 (Pyomo)
- **LP** (실수 최적화) / **IP** (정수 최적화) 선택
- 기간별 생산량, 재고, 인력, 고용/해고, 외주, 초과근무 산출
- 총비용 및 8개 항목 비용 구성 분해

### 4️⃣ 시각화 대시보드
- KPI 카드: 총비용, 총부족재고, 평균재고, 계획기간
- 수요·생산·재고 추이 차트 (라인+바 복합)
- 기간별 비용 구성 누적 막대 + 파이 차트
- 인력 변동 추이 차트
- 결품/재고 과다 이상 구간 자동 하이라이트

### 5️⃣ 결과 테이블
- 기간별 전체 결과 테이블 (결품 행 빨간색 강조)
- 기간별 비용 상세 테이블 (합계 행 포함)
- CSV 다운로드

### 6️⃣ 시나리오 관리
- 현재 계획을 이름·메모와 함께 JSON 저장
- 저장된 시나리오 불러오기 / 삭제
- 기존 시나리오 복제 후 수요 변경 재계산

### 7️⃣ 시나리오 비교
- 2개 이상 시나리오 선택해 KPI 테이블 비교
- 총비용 비교 막대 차트
- 시나리오별 재고 추이 / 결품 비교 차트

## 📐 최적화 모델

강의자료의 수식을 그대로 구현했습니다.

### 결정변수

| 변수 | 의미 | 단위 |
|------|------|------|
| $W_t$ | t월 종업원 수 | 인/월 |
| $H_t$ | t월 신규 고용 수 | 인/월 |
| $L_t$ | t월 해고 수 | 인/월 |
| $P_t$ | t월 생산량 | 개/월 |
| $I_t$ | t월 말 재고 | 개 |
| $S_t$ | t월 말 부족재고 | 개 |
| $C_t$ | t월 외주량 | 개/월 |
| $O_t$ | t월 총 초과시간 | Hr/월 |

### 목적함수 (총비용 최소화)

$$Z = \sum_t \left( w_{wage} \cdot W_t + w_{ot} \cdot O_t + w_H \cdot H_t + w_L \cdot L_t + w_I \cdot I_t + w_S \cdot S_t + w_P \cdot P_t + w_C \cdot C_t \right)$$

### 제약조건

| 제약 | 수식 |
|------|------|
| 노동력 균형 | $W_t = W_{t-1} + H_t - L_t$ |
| 생산능력 | $P_t \leq \frac{H_{day} \cdot D_{day}}{T_{std}} W_t + \frac{O_t}{T_{std}}$ |
| 재고균형 | $I_t = I_{t-1} + P_t + C_t - D_t - S_{t-1} + S_t$ |
| 초과근무 상한 | $O_t \leq OT_{max} \cdot W_t$ |
| 비음수 | $W_t, H_t, L_t, P_t, I_t, S_t, C_t, O_t \geq 0$ |

## 🔧 트러블슈팅

**Q. `SolverFactory` 오류가 발생해요**  
→ GLPK가 설치되어 있지 않거나 PATH에 없는 경우입니다. conda 환경에서 `conda install -c conda-forge glpk`로 설치하세요.

**Q. 계획 수립 실행 시 "풀이 실패" 메시지가 나와요**  
→ 수요 대비 초기 인력이 너무 적거나 목표 재고가 너무 높으면 실현 불가능한 문제가 됩니다. 파라미터를 조정해보세요.

**Q. 시나리오가 저장되지 않아요**  
→ `app.py`가 있는 폴더에 쓰기 권한이 있는지 확인하세요. `scenarios.json`이 같은 폴더에 자동 생성됩니다.
